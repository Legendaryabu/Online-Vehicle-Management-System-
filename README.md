# Online Vehicle Service Management System (OVSMS)

A comprehensive C#-based desktop application designed for streamlined vehicle service operations. This system implements a role-based access control (RBAC) architecture, providing distinct functionalities and interfaces for both Customers and Service Providers.

## 🚀 Key Features
* **Dual-Portal Access:** Specific login logic and interfaces for Customers and Service Providers.
* **Service Lifecycle Management:** End-to-end tracking of vehicle status from intake to completion.
* **Modular Architecture:** Clean file structures and database logic for easy maintenance and scalability.
* **Comprehensive Documentation:** Includes a detailed user manual to guide system operation.

## 🛠️ Tech Stack
* **Language:** C#
* **Platform:** Windows Desktop (WinForms/C#)
* **Logic:** Object-Oriented Programming (OOP) with role-based permissions.

## 📂 Project Highlights
* **User-Centric Design:** Intuitive UI for booking and managing services.
* **Database Reliability:** Robust handling of user profiles and vehicle service histories.
